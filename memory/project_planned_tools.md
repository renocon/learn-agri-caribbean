**Live tools (built):**
1. **Reference Glossary** (`docs/glossary/`) — 110 searchable CSEC/CVQ terms, 9 categories.
2. **Weedicide Rate Calculator** (`docs/weedicide-calc/`) — converts mL/ha or L/ha label rates to home garden quantities; supports knapsack, hand-pump, spray bottle; kitchen teaspoon equivalents; one-time safety modal.

**Planned tools (next up):**
3. **Crop Calendar / Planner** (`docs/crop-calendar/`) — visual planting timeline for Caribbean climate.
4. **Grow Box Simulator** (`docs/grow-box/`) — layout and planning tool for grow box operations.
5. **Tomato Growth Simulator** (`docs/tomato-sim/`) — stage-by-stage plant growth with disease demo and autoplay.

**Future (not yet scoped):**
6. **Nutrients Learning Tool** (`docs/nutrients/`) — interactive learning about plant nutrients; not yet framed.

**Why:** These were chosen as classroom demo tools for CSEC/CVQ teachers. Order above reflects rough build priority.
**How to apply:** When building a tool, use the folder name convention above. Always update index.html, sitemap.xml, and README.md after building.
